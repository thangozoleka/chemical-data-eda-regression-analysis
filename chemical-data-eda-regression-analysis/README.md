# Chemical Data EDA and Regression Analysis

## Overview
This project performs exploratory data analysis on chemical measurement data by comparing old and new imputed element values. The analysis uses regression modelling, R-squared scores, and visual diagnostics to assess consistency between measurement methods.

## Objectives
- Identify common chemical elements between old and new imputed datasets
- Compare measurement consistency using linear regression
- Evaluate agreement using slope, intercept, R-squared, p-values, and standard error
- Generate visual summaries for data quality assessment

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
